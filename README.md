# Open WebUI Lite

A lightweight, modern web interface chat App built with React, Vite, and Tailwind CSS for OpenAI compatible APIsï¼Œ inspired by [Open WebUI](https://github.com/open-webui/open-webui).

## Features

- Modern React-based UI
- Fast development with Vite
- Responsive design with Tailwind CSS. (in progress)
- Clean and intuitive user interface
- Component-based architecture
- Lightweight and efficient

## Prerequisites

- Node.js (v18 or higher)
- npm or yarn

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure your OpenAI API key in the `.env` file.(unsupported yet, in progress)
4. Start the development server:
   ```bash
   npm run dev
   ```
